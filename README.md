# Landing Page - Boda de Ángel y Erika

Landing page moderna y responsive para la boda de Ángel y Erika, que se celebrará el 17 de octubre de 2026.

## Características

- ✅ Diseño moderno y elegante
- ✅ Completamente responsive (optimizado para móviles)
- ✅ Contador de días restantes en tiempo real
- ✅ Secciones visuales con imágenes y texto
- ✅ Cabecera con foto de novios en blanco y negro
- ✅ Sin dependencias externas (HTML, CSS y JavaScript vanilla)

## Estructura del Proyecto

```
our-wedding-page/
├── index.html          # Página principal
├── styles.css          # Estilos
├── script.js           # Contador de días
├── README.md           # Este archivo
└── images/             # Carpeta para imágenes
    ├── novios.jpg      # Foto de los novios (cabecera)
    ├── lugar.jpg       # Foto del lugar
    ├── fecha.jpg       # Imagen para sección fecha
    └── confirmacion.jpg # Imagen para sección confirmación
```

## Cómo usar

1. Abre `index.html` en tu navegador
2. Reemplaza las imágenes en la carpeta `images/` con tus propias fotos
3. Personaliza los textos en `index.html` según tus necesidades

## Personalización

### Imágenes necesarias

Necesitarás las siguientes imágenes en la carpeta `images/`:

- **novios.jpg**: Foto de los novios para la cabecera (recomendado: formato horizontal, alta resolución)
- **lugar.jpg**: Foto de la finca o lugar donde se celebrará la boda
- **fecha.jpg**: Imagen decorativa para la sección de fecha
- **confirmacion.jpg**: Imagen decorativa para la sección de confirmación

### Textos a personalizar

En `index.html` puedes editar:

- Dirección del lugar
- Horarios de ceremonia y celebración
- Información de contacto
- Fecha límite de confirmación
- Cualquier otro texto descriptivo

## Notas

- El contador se actualiza automáticamente cada segundo
- La fecha objetivo está configurada para el 17 de octubre de 2026 a las 12:00
- Puedes cambiar la hora objetivo editando la fecha en `script.js`
